# HuVa Pivot — Wireframe v1

Static wireframe of the proposed Career Pivot Agent for huva.io/pivot.

**Live preview**: https://nickbrutyan-folder.github.io/huva-pivot-wireframe/

This is a click-through prototype — mock data only. Built to validate the experience flow before any real code lands.

## What it demonstrates

- The reframed hook ("Will Claude take your job?") instead of "paste your CV"
- An instant diagnostic anchored in real Anthropic Economic Index data
- The CV upload comes *after* the user is hooked, not as a barrier
- A scannable output — diagnosis, 5 ranked pivots, 30-day plan, LinkedIn rewrite, 3 portfolio projects
- A shareable AI-augmented identity card at the end (LinkedIn-ready)
- Soft, single HuVa Academy CTA — no pop-ups, no email gates

## Try it as different personas

Use the **Preview as** dropdown in the top bar to switch between:
- Marketing Manager (low risk pivot)
- Software Engineer (high exposure, multiple pivot lanes)
- Customer Service Rep (highest risk in 2026 data)

Each persona has its own diagnostic, pivots, plan, LinkedIn copy, and projects.
